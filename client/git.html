Git Cheatsheet

Command Description Notes
Most Useful
git status Show the status of the working directory
git stash Stash changes temporarile Hint... You will love this!
git stash pop Regain stashed changes Hint... You will love this even more!!!
GIT config
git config --global user.name "[name]" Set the name you want attached to your commits
git config --global user.email "[email address]" Set the email you want attached to your commits
git config --global color.ui auto Enable helpful colorization of command line output [normal, black, red, green, yellow, blue, magenta, cyan, white]
git config --global core.editor "[editor]" Set the text editor used by commands for all users on the system
git config --global core.autocrlf input Configure Git to properly handle line endings
GIT Credentials
git config credential.helper store Save credentials in memory for use by future Git programs
git config --global credential.helper cache Save credentials in memory for use by future Git programs
git config --global credential.helper 'cache --timeout=3600' Save credentials in memory for use by future Git programs
Initializing
git init Initialize a new Git repository
git clone [repository_url] Clone a remote repository
git remote add origin [repository_url] Add a remote repository
git add [file] Add a file to the staging area Can also use a "." for to add all files not mentioned in the .gitignore files
Basics
git status Show the status of the working directory
git commit -m "[message]" Commit changes with a message
git push Push changes to a remote repository
git pull Pull changes from a remote repository
git branch List all branches
git checkout [branch] Switch to a different branch
git merge [branch] Merge changes from a branch
git log Show commit history
git diff Show changes between commits, commit and working tree, etc
git reset [file] Unstage a file
git rm [file] Remove a file from the working directory and staging area
git mv [file] Move or rename a file
git tag [tag] Create a tag for a specific commit
git fetch Download objects and refs from another repository
git remote -v Show remote repositories
git remote show [remote] Show information about a remote
git remote update Fetch updates from a remote
git remote remove [remote] Remove a remote
git remote rename [old_name] [new_name] Rename a remote
git remote set-url [remote] [url] Change the URL of a remote
