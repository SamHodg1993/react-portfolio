JavaScript Cheatsheet

Code Snippet Description Notes
Variables and Data Types
let x = 5; Declare a variable with let Use let for block-scoped variables
const y = 10; Declare a constant variable Constants cannot be reassigned
var z = true; Declare a variable with var var has function or global scope
Strings
let str = "Hello"; Create a string
str.length; Get the length of a string
str.toUpperCase(); Convert a string to uppercase
str.toLowerCase(); Convert a string to lowercase
str.includes("ell"); Check if a string contains a substring
Arrays
let arr = [1, 2, 3]; Create an array
arr.length; Get the length of an array
arr.push(4); Add an element to the end of an array
arr.pop(); Remove the last element from an array
arr.forEach(function(item) {...}); Iterate over an array
Functions
function greet(name) {
  console.log("Hello, " + name);
} 		Define a function
greet("John"); Call a function
const square = (x) => x * x;
console.log(square(5)); 		Arrow function Shorthand for defining functions
DOM Manipulation
document.getElementById("id"); Get an element by its ID
document.getElementsByClassName("class"); Get elements by their class name
element.innerHTML = "Hello, World!"; Set the inner HTML of an element
element.style.color = "red"; Set the style of an element
element.addEventListener("click", function() {...}); Add an event listener to an element
