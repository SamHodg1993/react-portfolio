Node.js with Express Cheatsheet

Code Snippet Description Notes
Getting Started
npm init Initialize a new Node.js project Creates a package.json file
npm install express Install Express.js Adds Express to your project dependencies
Basic Express App
const express = require('express');
const app = express(); 		Import Express and create an app instance
app.get('/', (req, res) => {
  res.send('Hello, World!');
}); 		Define a route for the root URL Responds with 'Hello, World!'
app.listen(3000, () => {...}); Start the server and listen on port 3000
Routing
app.get('/users', ...) Define a GET route for '/users'
app.post('/users', ...) Define a POST route for '/users'
app.use('/api', router) Mount a router for '/api' routes
Middleware
app.use(express.json()) Parse incoming JSON data
app.use(express.static('public')) Serve static files from the 'public' directory
app.use((req, res, next) => {
  // middleware logic
  next();
}); 		Create a custom middleware function Don't forget to call next()
Handling Requests
req.params Access route parameters e.g., /users/:id
req.query Access query string parameters e.g., /search?q=hello
req.body Access request body data Requires middleware like express.json()
res.json(data) Send JSON data as a response
res.status(404).send('Not found') Send a custom response with a status code
