Linux Commands Cheatsheet

Command Description Example
Useful
history||grep ssh Search for previous used commands this is searching for commands containing "ssh" Super useful for getting ip's etc...
scp -r [fileNames||dirName/*] [username]@[url||IP]:/path/to/servers/target/dir Sends files from one server to another or your pc to a server using the ssh functionality! Super useful for getting files like ssl certificates to a server...
sudo lsof -t:8080 Lists services that are currently using the reference port. Alternatively you could use sudo lsof -i -P -n | grep LISTEN
Navigation
pwd Print the current working directory pwd
cd Change directory cd /path/to/directory
ls List files and directories ls -l (long listing)
File Operations
cp Copy files and directories cp file1.txt file2.txt
mv Move or rename files and directories mv file.txt new_file.txt
rm Remove files and directories rm file.txt
Text Manipulation
cat Concatenate and print files cat file.txt
grep Search for patterns in files grep pattern file.txt
sed Stream editor for editing files sed 's/old/new/g' file.txt
Process Management
ps List running processes ps aux
kill Terminate a process kill process_id
top Display system resource usage top
Permissions
chmod Change file permissions chmod +x script.sh
chown Change file ownership chown user:group file.txt
System Information
uname Print system information uname -a
df Display disk space usage df -h
du Display disk usage for files and directories du -sh /path/to/directory
