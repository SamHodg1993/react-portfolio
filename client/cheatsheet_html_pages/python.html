Python Cheatsheet

Code Snippet Description Notes
Variables and Data Types
x = 5 Assign a value to a variable Python is dynamically typed
x = "Hello" Assign a string to a variable
x = True Assign a boolean value to a variable
Control Structures
if x > 0:
  print("x is positive")
elif x < 0:
  print("x is negative")
else:
  print("x is zero") 		Conditional statement (if/elif/else) Note the indentation
for i in range(5):
  print(i) 		For loop Prints 0, 1, 2, 3, 4
while x < 10:
  print(x)
  x += 1 		While loop Prints 0, 1, 2, ..., 9
Functions
def greet(name):
  print("Hello, " + name) 		Define a function
greet("John") Call a function Prints "Hello, John"
def add(a, b):
  return a + b 		Function with return value
Lists and Dictionaries
numbers = [1, 2, 3, 4, 5] Create a list
numbers.append(6) Add an element to a list
person = {"name": "John", "age": 30} Create a dictionary
person["city"] = "New York" Add a key-value pair to a dictionary
File Operations
file = open("data.txt", "r")
contents = file.read()
file.close() 		Read the contents of a file "r" for read mode
file = open("output.txt", "w")
file.write("Hello, World!")
file.close() 		Write to a file "w" for write mode
with open("data.txt", "r") as file:
  contents = file.read() 		Read a file using a context manager File closed automatically
