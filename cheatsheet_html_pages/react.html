React Cheatsheet

Code Snippet Description Notes
Components
import React from 'react';

const MyComponent = () => {
  return <div>Hello, World!</div>;
};

export default MyComponent; 		Create a functional component Functional components are simple functions
import React, { Component } from 'react';

class MyComponent extends Component {
  render() {
    return <div>Hello, World!</div>;
  }
}

export default MyComponent; 		Create a class component Class components have state and lifecycle methods
JSX
<div>Hello, World!</div> JSX element JSX is a syntax extension for React
{variable} Render a JavaScript variable Use curly braces to embed expressions in JSX
<MyComponent prop={value} /> Pass props to a component Props are read-only and passed down from parent components
State and Lifecycle
constructor(props) {
  super(props);
  this.state = { count: 0 };
} 		Initialize state in a class component State is mutable and private to a component
this.setState({ count: this.state.count + 1 }) Update state Always use setState to update state
componentDidMount() {
  // code to run after component mounts
} 		Lifecycle method componentDidMount is called once after the initial render
Hooks
import React, { useState } from 'react';

const MyComponent = () => {
  const [count, setCount] = useState(0);
  // ...
}; 		Use the useState hook Hooks allow functional components to have state
import React, { useEffect } from 'react';

const MyComponent = () => {
  useEffect(() => {
    // code to run after component mounts
  }, []);
  // ...
}; 		Use the useEffect hook useEffect allows functional components to have lifecycle methods
