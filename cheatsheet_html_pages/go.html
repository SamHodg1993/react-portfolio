C++ Notes

Code Snippet Outcome Notes
Basics
fmt.Sprintf("%v", 123) "123" Default format for the value.
fmt.Sprintf("%+v", struct{Name string}{"John"}) "{Name:John}" Prints the field names in a struct.
fmt.Sprintf("%#v", struct{Name string}{"John"}) `struct { Name string }{Name:"John"}` Go-syntax representation of the value.
fmt.Sprintf("%T", 123) "int" Type of the value.
fmt.Sprintf("%%") "%" A literal percent sign.
String and Character Formatting
fmt.Sprintf("%s", "hello") "hello" Interprets the value as a plain string.
fmt.Sprintf("%q", "hello") "\"hello\"" Double-quoted string safely escaped with Go syntax.
fmt.Sprintf("%x", "hello") "68656c6c6f" Base 16, with lower-case letters for a-f.
fmt.Sprintf("%X", "hello") "68656C6C6F" Base 16, with upper-case letters for A-F.
Integer Formatting
fmt.Sprintf("%d", 123) "123" Base 10.
fmt.Sprintf("%b", 7) "111" Base 2.
fmt.Sprintf("%o", 8) "10" Base 8.
fmt.Sprintf("%x", 255) "ff" Base 16, with lower-case letters for a-f.
fmt.Sprintf("%X", 255) "FF" Base 16, with upper-case letters for A-F.
Floating Point and Complex Formatting
fmt.Sprintf("%f", 123.456) "123.456000" Decimal point but no exponent.
fmt.Sprintf("%e", 123.456) "1.234560e+02" Scientific notation with 'e' for exponent.
fmt.Sprintf("%E", 123.456) "1.234560E+02" Scientific notation with 'E' for exponent.
fmt.Sprintf("%g", 123.456) "123.456" Uses the shortest representation: %e or %f.
fmt.Sprintf("%G", 123.456) "123.456" Uses the shortest representation: %E or %f.
Width and Precision
fmt.Sprintf("%6d", 123) " 123" Minimum number of characters to be printed.
fmt.Sprintf("%.2f", 123.456) "123.46" Number of digits after the decimal point for floating point.
fmt.Sprintf("%6.2f", 123.456) "123.46" Width and precision together.
